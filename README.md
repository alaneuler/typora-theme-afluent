# Typora Theme: AFluent

This is a theme carefully crafted & designed for Typora, based on [Fluent-Typora](https://github.com/li3zhen1/Fluent-Typora). This theme has a built-in dark mode support.

## Formattings
You can place headers in blockquote with extra styles.

```markdown
> # Bookmark
> This is `H1` inside a block quote.
```

```markdown
> ###### Question
> This is `H6` inside a block quote.
```
